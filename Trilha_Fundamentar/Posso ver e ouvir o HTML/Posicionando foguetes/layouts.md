## Posicionamentos

Controlar onde, na página, o elemento irá ficar, alterando o fluxo normal dos elementos.

- Name: position
- Value: static|relative|absolute|fixed ( Por padrão é static)

## Relative

- top, right, bottom, left, z-index

## Absolute

- top, right, bottom, left, z-index

## Fixed

- top, right, bottom, left, z-index

## Element Stacking

- z-index
